KorewaCraft

KorewaCraft was a 1.16.3 Minecraft server created by WeebRyfried and developed by nekotosume. Used to run on donations but decided to close the project down due to low player presence.

A chaotic city

The city, called by the players "Retard Town" (lol) is the server spawn point and everything built there is built by players in Survival Mode.
